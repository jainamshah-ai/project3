# Student Data Organizer

## Project Description

The **Student Data Organizer** is a simple Python console-based application used to manage student records.
This program allows users to:

* Add student details
* Display all student records
* Update student information
* Delete student records
* Display all subjects offered
* Exit the program safely

The project is created using basic Python concepts such as:

* Lists
* Dictionaries
* Loops
* Conditional statements
* User input handling

---

## Features

### 1. Add Student

Users can add:

* Student ID
* Name
* Age
* Grade
* Date of Birth
* Subjects

---

### 2. Display All Students

Displays all stored student records in a formatted way.

---

### 3. Update Student Information

Allows updating student details using Student ID.

---

### 4. Delete Student

Removes a student record using Student ID.

---

### 5. Display Subjects Offered

Shows all unique subjects available in the student records.

---

### 6. Exit Program

Safely exits the application.

---

## Data Structure Used

Used to store multiple student records.

### Dictionary

Used to store individual student information.

Example:

```python
student = {
    "id": student_id,
    "name": name,
    "age": age,
    "grade": grade,
    "dob": dob,
    "subjects": subjects

```

## How to Run the Project

### Step 1:

Install Python 3 on your system.

### Step 2:

Save the file as:

```bash
project3.py
```

### Step 3:

Open terminal or command prompt.

### Step 4:

Run the program:

```bash
python project3.py
```
